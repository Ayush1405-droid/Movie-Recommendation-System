# Movie Recommendation System

This project is a Machine Learning-based Movie Recommendation System developed using Python. It uses content-based filtering to recommend movies based on genre, keywords, and overview. The system processes movie data using Pandas, converts text into numerical vectors using CountVectorizer, and calculates similarity using cosine similarity. A user-friendly Tkinter GUI allows users to enter a movie name and receive top recommended movies instantly.

## Technologies Used

* Python
* Pandas
* Scikit-learn
* Tkinter
* CountVectorizer
* Cosine Similarity

## Features

* Recommends top 5 similar movies
* Simple graphical user interface
* Fast and efficient recommendation system
* Error handling for invalid movie names
